Comment contrecarrer l'espionnage des géants malicieux du Web (Google, etc.)

Published: jeu. 13 février 2025
By LostLevels94

In Guide.

Google, Microsoft, Apple etc. vous pistent en permanence. Ils prennent toutes les données qu'ils peuvent sur vous et c'est d'ailleurs pour cela qu'on peut les qualifier de logiciels privateurs/malwares. Heureusement, vous pouvez utiliser d'autres services similaires. Voici une liste (non exhaustive) des applis les plus utilisées et leurs alternatives les plus sérieuses.

Logiciels et services

  • Google Search : DuckDuckGo
  • YouTube : PeerTube
  • Client YouTube : FreeTube sur ordi/NewPipe sur Android
  • Google Maps : HERE WeGo
  • Google Mail : Infomaniak Mail
  • Google Earth : KDE Marble
  • Microsoft 365 Office : Combo LibreOffice X ONLYOFFICE
  • Google Docs : Infomaniak Docs
  • iMovie : KDenlive
  • Github : Gitlab
  • Google Photos : Ente
  • Google Chrome : Vivaldi Browser sur ordi/Cromite sur mobile
  • Google Keep : Joplin/StandardNote
  • Google Drive/iCloud : kDrive
  • Lien avec Windows : KDE Connect
  • Adobe Photoshop : GIMP
  • WhatsApp : Signal
  • Google Play Store : F-Droid
  • Client Google Play Store : App Lounge
  • Teams : kMeet
  • X : Mastodon
  • Paint : kolourpaint
  • Outlook : Geary
  • weTransfer : SwissTransfer
  • Microsoft Visual Studio : Geany
  • Google Actualités : Feedly
  • Avast : ClamAV
  • Google Caméra : OpenCamera
  • Google One : Murena Workspace
  • Google Passwords : Bitwarden
  • QRCo.de : Infomaniak Chk

OS

  • Windows/macOS : Fedora, Linux Mint ou Ubuntu
  • Android avec ROM propriétaire : LineageOS (ou /e/OS qui en est un fork)

links

  • Python.org
